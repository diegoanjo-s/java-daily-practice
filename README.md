# ☕ Java Daily Practice

Exercícios diários de Java, organizados por tema — do básico à orientação a objetos — como parte da minha jornada de transição de carreira para desenvolvedor backend.

---

## 🎯 Sobre este repositório

Depois de 13+ anos em gestão operacional (Arcos Dourados/McDonald's, liderando equipes de até 30 pessoas), decidi migrar pra área de tecnologia. Esse repositório documenta minha prática diária de Java: um tema novo por dia, sempre com exercícios aplicados a contextos reais que eu já conheço bem — operação, vendas, gestão de equipe.

A ideia é simples: **consistência gera domínio**. Cada pasta aqui é um dia de estudo, seguindo a sequência oficial da [Maratona Java Virado no Jiraya](https://www.youtube.com/playlist?list=PL62G310vn6nFIsOCC0H-C2infYgwm8SWW) (DevDojo), complementada pelo Santander Bootcamp 2026 (trilha Java Backend).

## 📚 Estrutura

Cada pasta representa um dia de prática, focado em um tema específico:

| Pasta | Tema | Status |
|---|---|---|
| `01` | Tipos primitivos + casting | ✅ |
| `02` | Operadores (aritméticos, relacionais, lógicos) | 🔜 |
| `03` | Condicionais (IF, IF ELSE, ternário) | 🔜 |
| `04` | Tabela verdade + SWITCH | 🔜 |
| `05` | Loops (WHILE, DO WHILE, FOR, break, continue) | 🔜 |
| `06` | Arrays (parte 1-4, foreach) | 🔜 |
| `07` | Arrays multidimensionais | 🔜 |
| `08` | OOP: introdução + coesão + exercício carro | 🔜 |
| `09` | OOP: referência, métodos, parâmetros, `this` | 🔜 |
| `10` | OOP: encapsulamento, get/set, sobrecarga | 🔜 |
| `11` | OOP: construtores → herança → polimorfismo → exceções | 🔜 |

> Status atualizado a cada commit. ✅ concluído · 🔵 em andamento · 🔜 planejado
